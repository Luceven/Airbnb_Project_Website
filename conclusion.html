Conclusion

The purpose of this Airbnb Pricing Analysis for New York City is to successfully predict the price of Airbnb based on features and data extracted from Yelp, Hotels and Airbnb listing. We used different approaches to explore data insight, such as text mining, association rules, in order to improve our prediction accuracy. Because of the limitation of data, the weakness of this paper is seasonality. In the future, our final goal is to provide the best price setting strategy for the landlord and the best Airbnb apartment recommendation for users.


More Data

Transportation Data

Transportation data was first planned to retrieve through Walkscore API, which provides a convenience evaluation of a given location, by returning a “Transit Score”. We’ve finished the Python script for requesting transit score data, but unfortunately the API key that we requested is not working and we’re still waiting for Walkscore’s response. As a substitution, we’ve prepared a script to request Google Maps API data, which provides organized json data like nearby bus/metro stations with user defined location and radius. Data will be selected by using longitude and latitude of the Airbnb houses to measure. As the project moves on, we will either get the new API key and request data or switch to Google Maps API.
